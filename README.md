# Battlesimulator
斗蛐蛐小游


一个基于 **Pygame** 的实时兵种对战模拟程序。红蓝两军在网格战场上自动移动、寻找目标并交战，直到一方全灭。

> ⚠️ **作者声明**：本项目仅供学习与非商业用途。转载或使用请注明出处。  
---

## 🎮 游戏特色

- **四种兵种**，各具特性：
  - **步兵（Infantry）**：圆形，高生命，近战。
  - **弓箭手（Archer）**：三角形，中等伤害，远程（3格）。
  - **骑兵（Cavalry）**：正方形，移动速度快（2倍），近战。
  - **炮兵（Artillery）**：六边形，高伤害高血量，超远程（5格），但移动缓慢。
- 自动寻路与目标锁定
- 实时生命值条显示
- 随机生成初始兵力（每方 10–20 个单位）
- 支持重新开始游戏

---

## 🖥️ 操作说明

| 按键 | 功能 |
|------|------|
| `S` | 开始模拟（初始界面需按 S 启动） |
| `R` | 重新开始一局 |
| 关闭窗口 | 退出程序 |

---

## 🛠️ 运行环境

- Python 3.7 或更高版本
- Pygame 库

### 安装依赖

```bash
pip install pygame
```
